# 🛒 E-commerce Sales Dashboard (Power BI)

## 📌 Overview
This project analyzes e-commerce sales data to track revenue, orders, product performance, and payment methods.

## 📊 Dashboard Preview
![Dashboard](dashboard.png)

## 🎯 Objective
To monitor business performance and identify key revenue drivers.

## 🛠 Tools Used
- Power BI  
- Power Query (ETL)  

## 🔄 Data Processing
- Cleaned and transformed data using Power Query  
- Created calculated columns and measures  
- Prepared data for analysis  

## 📊 Dashboard Features
- KPI metrics (Revenue, Orders, AOV)  
- Product and category performance analysis  
- Payment method distribution  
- Monthly sales trends  
- Interactive filters  

## 📈 Key Insights
- Cash on Delivery is the most used payment method  
- Top products contribute majority of revenue  
- Sales show monthly variation  

## 📁 Files Included
- Power BI Dashboard (.pbix)  
- Dataset (CSV/Excel)
