# 📊 Excel Sales Performance Dashboard

## 🔍 Overview

This project is an interactive Sales Performance Dashboard built using Microsoft Excel.
It provides clear insights into sales trends, category-wise distribution, city-wise performance, and key business KPIs.

---

## 📌 Key Features

* KPI Cards for:

  * Total Sales
  * Average Order Value (AOV)
  * Total Quantity
  * Total Orders
* Monthly Sales Trend line chart
* City-wise sales comparison
* Category-wise sales distribution
* Interactive slicers for dynamic filtering
* Clean one-screen dashboard layout

---

## 📈 Key Insights

* Highest sales observed in **April and July**
* Lowest sales recorded in **February**
* **Fashion** category contributes the highest revenue
* **Delhi** shows the strongest city-wise sales performance
