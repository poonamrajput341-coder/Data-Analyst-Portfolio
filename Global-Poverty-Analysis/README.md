# 🌍 Global Poverty & Inequality Analysis (Power BI)

## 📊 Dashboard Preview
![Dashboard](dashboard.png)

## 📌 Overview
This project analyzes global poverty trends and income inequality across countries from 1980 to 2024 using Power BI.

## 🎯 Objective
To understand the relationship between poverty rate and income inequality (Gini Index) and identify global trends.

## 🛠 Tools Used
- Power BI  
- Power Query (ETL)  

## 🔄 Data Processing
- Cleaned and transformed data using Power Query  
- Handled missing values and corrected data types  
- Standardized country and year data  

## 📊 Dashboard Features
- Poverty trend analysis over time  
- Correlation analysis between poverty and Gini Index  
- Country-wise comparison  
- Interactive slicers (Year, Country)  

## 📈 Key Insights
- Global poverty rate has decreased over time  
- Strong positive correlation (0.75) between inequality and poverty  
- Countries with higher inequality show higher poverty rates
