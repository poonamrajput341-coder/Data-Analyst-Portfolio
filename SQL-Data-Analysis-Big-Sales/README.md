# 📊 SQL Data Analysis - Big Sales Dataset

## 📌 Overview
This project performs data analysis on a retail sales dataset using SQL. 
The goal is to extract meaningful business insights such as top-performing products, cities, and sales trends.

---

## 🛠️ Tools Used
- MySQL
- SQL
- MySQL Workbench

---

## 📂 Dataset
The dataset contains:
- Product
- Category
- Price
- Quantity
- Total Amount
- City
- Date

---

## 🔍 Key Analysis Performed

### 1. Top Cities by Total Sales
Identified cities generating the highest revenue.

### 2. Top Selling Product by Quantity
Found the most sold product based on quantity.

### 3. Product-wise Sales Analysis
Analyzed total and average sales per product.

### 4. Category-wise Revenue Analysis
Compared revenue and average price across categories.

### 5. Daily Sales Trend
Analyzed how sales change over time.

### 6. High Value Transactions
Filtered transactions above 10,000.

### 7. Sales by Category and City
Analyzed combined performance across category and location.
