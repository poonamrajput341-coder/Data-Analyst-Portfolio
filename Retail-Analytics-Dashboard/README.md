# 🛍 Retail Analytics Dashboard (Power BI)

## 📌 Overview
This project provides insights into retail sales performance across categories, customers, and time.

## 📊 Dashboard Preview
![Dashboard](dashboard.png)

## 🎯 Objective
To analyze sales trends, track KPIs, and identify top-performing categories.

## 🛠 Tools Used
- Power BI  
- Power Query (ETL)  

## 🔄 Data Processing
- Performed data cleaning and transformation using Power Query  
- Removed duplicates and handled missing values  
- Structured data for reporting  

## 📊 Dashboard Features
- KPI cards (Total Sales, Orders, Customers)  
- Category-wise sales analysis  
- Daily sales trend visualization  
- Interactive filters (Date, Category)  

## 📈 Key Insights
- Fashion category generates highest revenue  
- Sales show stable trends over time  
- Accessories and Electronics are competitive categories
